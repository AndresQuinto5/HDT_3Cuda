# Hoja de trabajo 3

Diseña programas paralelos en computadoras heterogéneas usando GPUs, CUDA y OpenACC

## How to Compile
```
nvcc hello.cu -o ./hello
nvcc hello2.cu -o ./hello2
nvcc vectoradd2.cu -o ./vectoradd2
```

## How to Run

Once you have compiled and used 
```
./hello
./hello2
./vectoradd2
```
